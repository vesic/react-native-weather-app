# Weather App

React Native demonstration purpose app

### Prerequisites

* Node

### Setup

Follow https://facebook.github.io/react-native/docs/getting-started for latest setup instructions

### Installing

```
git clone repo
cd repo
yarn|npm install
react-native run-ios|run-android
```
